

# 简易版飞书待办系统需求文档（PRD）
## 一、项目概述
### 1.1 产品定位
基于飞书待办的核心设计逻辑，打造一款**轻量、易用、高效**的简易版待办事项管理系统。聚焦个人及小型团队的核心任务管理需求，去除复杂冗余功能，保留“任务跟踪、分类管理、简易协作”三大核心能力，帮助用户实现“事事有记录、进度有跟踪、协作无阻碍”。

### 1.2 目标用户
- 个人用户：需要梳理日常工作/生活任务、避免遗忘的普通用户；
- 小型团队（3-20人）：需简单分配任务、同步进度的小团队（如项目小组、部门小组）。

### 1.3 核心价值
- 操作极简：无需复杂配置，快速上手创建和管理任务；
- 功能聚焦：保留飞书待办核心实用功能，无冗余干扰；
- 轻量协作：支持小型团队任务共享与进度同步，降低沟通成本。

## 二、功能需求
### 2.1 任务管理模块（核心）
#### 2.1.1 任务创建
- 必填项：任务标题（限50字内）、任务状态（默认“待处理”）；
- 选填项：截止时间（支持日期+时间选择）、优先级（高/中/低，默认“中”）、详细描述（限500字内）；
- 扩展项：支持添加1个子任务层级（最多5个子任务）、设置重复任务（仅支持按日/周/月固定周期）。

#### 2.1.2 任务操作
- 基础操作：支持编辑任务信息、删除任务（仅创建人/负责人可操作）、标记任务状态变更（待处理→进行中→已完成）；
- 子任务操作：子任务支持独立标记完成/未完成，所有子任务完成后，父任务可自动标记为“已完成”（可手动关闭该规则）；
- 附件支持：支持上传10MB以内的文档、图片附件（最多3个），方便补充任务上下文。

### 2.2 分类与视图模块
#### 2.2.1 任务清单（核心分类）
- 支持创建自定义任务清单（如“工作任务”“个人事务”“项目A”），最多创建10个清单；
- 任务可自由移动至不同清单，单个任务仅支持归属1个清单；
- 支持清单重命名、删除（删除清单不删除任务，任务自动归为“未分类”）。

#### 2.2.2 视图与筛选
- 支持2种基础视图：列表视图（默认，按创建时间排序）、看板视图（按任务状态分组展示）；
- 筛选功能：支持按“优先级、截止时间、状态、创建人”筛选任务（仅支持“且”逻辑）；
- 排序功能：支持按“截止时间（升序/降序）、优先级（高→低）、创建时间”手动切换排序规则。

### 2.3 协作功能模块（简易版）
#### 2.3.1 清单协作
- 支持邀请协作人加入任务清单（通过手机号/用户名搜索），最多邀请15名协作人；
- 权限设置：仅2种权限（可阅读/可编辑），创建人默认拥有最高权限，可修改协作人权限或移除协作人；
  - 可阅读：查看清单及任务、添加任务评论，仅能编辑自己负责的任务；
  - 可编辑：编辑清单名称、移动/添加/删除清单内任务、修改任务信息（不含删除他人创建的任务）；
- 协作通知：任务被指派、评论、状态变更时，相关协作人会收到系统通知。

#### 2.3.2 任务指派
- 创建任务时可指定1名负责人（默认为创建人），支持后续变更负责人；
- 支持添加“关注人”（最多3人），关注人会收到任务状态变更通知，仅可查看任务和评论。

### 2.4 提醒与通知模块
- 截止时间提醒：任务截止前1小时自动发送系统通知（支持在“设置”中关闭）；
- 逾期提醒：任务逾期后，每24小时发送1次提醒，直至任务标记为“已完成”；
- 协作通知：收到任务指派、评论回复时，实时推送系统通知。

### 2.5 数据管理模块
- 任务归档：已完成的任务支持手动归档，归档后从列表中隐藏，可在“归档中心”查看或恢复；
- 数据导出：支持将单个清单的任务导出为Excel文件（包含任务标题、状态、负责人、截止时间等核心字段）；
- 本地存储：个人任务数据自动本地备份，团队清单数据云端同步（简易版不支持数据恢复功能）。

## 三、非功能需求
### 3.1 性能要求
- 页面加载时间≤1秒，任务操作（创建/编辑/标记完成）响应时间≤500ms；
- 支持同时在线人数≤100人，单个清单最多容纳500条任务。

### 3.2 兼容性要求
- 支持主流浏览器（Chrome、Edge、Firefox 最新3个版本）；
- 支持移动端自适应（手机浏览器访问可正常操作核心功能，无需单独开发APP）。

### 3.3 易用性要求
- 操作流程极简：创建任务步骤≤3步，切换视图、筛选任务操作≤2步；
- 界面设计：遵循飞书简约风格，色彩以白/浅灰为主，优先级用红（高）、黄（中）、绿（低）区分，视觉清晰无干扰；
- 无强制引导：不添加广告、不强制推送无关功能，专注任务管理核心场景。

## 四、界面原型建议（核心页面）
### 4.1 首页（任务总览）
- 左侧：清单导航栏（显示所有清单及未完成任务数）、“归档中心”入口、设置入口；
- 中间：任务视图区（默认列表视图，顶部为筛选/排序工具栏、“新建任务”按钮）；
- 右侧：协作人列表（仅团队清单显示，可快速查看清单内协作人及权限）。

### 4.2 任务创建/编辑弹窗
- 顶部：标题输入框；
- 中部：核心字段区域（截止时间、优先级、负责人/关注人选择器）；
- 底部：详细描述输入框、附件上传入口、“确认创建/保存”按钮。

### 4.3 清单协作设置页
- 协作人管理区：显示已添加的协作人及权限，支持添加/移除/修改权限；
- 权限说明区：简洁说明“可阅读/可编辑”权限的核心差异；
- 分享链接区：支持生成清单分享链接（默认有效期7天，可手动关闭分享）。

## 五、简易版与飞书原版差异说明
| 功能维度   | 飞书原版待办                         | 简易版待办系统               |
| ---------- | ------------------------------------ | ---------------------------- |
| 视图类型   | 列表/看板/甘特图/多维表格等          | 仅保留列表/看板2种基础视图   |
| 协作权限   | 多级别权限（所有者/可编辑/可阅读等） | 仅2种权限（可编辑/可阅读）   |
| 任务复杂度 | 支持多层级子任务、自定义字段         | 仅1级子任务，固定核心字段    |
| 数据功能   | 数据统计、仪表盘分析                 | 仅支持基础Excel导出          |
| 集成能力   | 与飞书文档/日历/群聊深度集成         | 无外部集成，专注独立任务管理 |
